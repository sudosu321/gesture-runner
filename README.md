# Gesture Runner Game

A simple game where the player jumps using hand gestures detected through a webcam.

## Controls

* Open hand → Jump
* Closed hand → Stay

## Features

* Real-time hand tracking
* Gesture-based control
* Endless runner gameplay

## Tech Stack

* Python
* OpenCV
* MediaPipe
* Pygame

## How to Run

1. Install dependencies:

   ```
   pip install pygame opencv-python mediapipe
   ```
2. Run the game:

   ```
   python game.py
   ```

## Description

This project demonstrates the use of computer vision to control a game without traditional input devices like a keyboard.
